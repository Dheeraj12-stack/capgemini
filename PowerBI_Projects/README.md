# 📊 Education Performance Analytics Dashboard

---

# 📌 Project Overview

This project presents an interactive Education Performance Analytics Dashboard developed using Power BI.

The dashboard provides detailed insights into:

* Student Performance
* Attendance Analysis
* Assignment Scores
* Course-wise Metrics
* Gender Distribution
* City-wise Student Distribution

The objective of this project is to convert raw educational data into meaningful visual insights for academic analysis and decision-making.

---

# 📈 Key Insights

* Identified courses with the highest student enrollments
* Analyzed attendance percentage across multiple courses
* Compared average exam scores by course
* Visualized gender distribution of students
* Tracked assignment score vs exam score relationships
* Analyzed study hours impact on exam performance
* Mapped city-wise student distributions geographically

---

# 🚀 Dashboard Features

* Interactive Filters and Slicers
* KPI Summary Cards
* Dynamic Visualizations
* Geographic Mapping
* Drill-down Analysis
* Multi-page Dashboard Navigation
